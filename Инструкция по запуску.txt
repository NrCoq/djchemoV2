запускается программа пользуясь коммандой 

python manage.py runserver

для того чтобы получить доступ к сайту (который запустится по умолчанию на localhost:8000)
требуется заранее создать пользователя с коммандой

python manage.py createsuperuser

django затем спросит вас указать имя админа и пароль.

после этого можно зайти в приложении. В нем одно окошко для введения структуры молекулы в форме SMILES нотации.
